2026-05-19 学习重点总结
=========================

1. 变量（变量.py）
   - 四种基本数据类型：int（整数）、float（浮点数）、str（字符串）、bool（布尔值）
   - type() 查看变量类型
   - 类型转换：int()、float()、str()、bool()
   - int() 的 base 参数可指定进制（2进制、8进制、16进制）
   - ord() 获取字符的 Unicode 编码

2. 运算符（运算符.py）
   - 算术运算：+ - * / //（整除） %（取余） **（幂）
   - 字符串运算：* 重复、+ 拼接
   - 进阶：<< 位左移、not 逻辑非、%d 格式化输出

3. 分支结构（分支结构.py）
   - if / elif / else：经典条件判断
   - match / case：模式匹配（Python 3.10+），可匹配具体值或使用 _ 通配符
   - 获取用户输入：input()

4. 循环结构（循环.py）
   - for-in + range()：计数循环，_ 作丢弃变量
   - while：条件循环
   - break 终止循环，continue 跳过本轮
   - 嵌套循环：九九乘法表示例，内层 range 依赖外层变量
   - time.sleep()：暂停执行，控制输出节奏

5. 综合实例：Craps 赌博游戏（分支和循环实例.py）
   - random.randrange() 生成随机数模拟骰子
   - 嵌套 while 实现游戏主循环 + 下注校验
   - 完整的状态流转：下注 → 摇骰 → 胜负判定 → 资产变化
   - if/elif 实现游戏规则（首轮 7/11 胜，2/3/12 负，其余进入循环）
   - f-string 动态输出当前资产和点数
