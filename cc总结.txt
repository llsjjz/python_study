2026-05-19 学习重点总结
=========================

1. 变量（变量.py）
   - 四种基本数据类型：int（整数）、float（浮点数）、str（字符串）、bool（布尔值）
   - type() 查看变量类型
   - 类型转换：int()、float()、str()、bool()
   - int() 的 base 参数可指定进制（2进制、8进制、16进制）
   - ord() 获取字符的 Unicode 编码

2. 运算符（运算符.py）
   - 算术运算：+ - * / //（整除） %（取余） **（幂）
   - 字符串运算：* 重复、+ 拼接
   - 进阶：<< 位左移、not 逻辑非、%d 格式化输出

3. 分支结构（分支结构.py）
   - if / elif / else：经典条件判断
   - match / case：模式匹配（Python 3.10+），可匹配具体值或使用 _ 通配符
   - 获取用户输入：input()

4. 循环结构（循环.py）
   - for-in + range()：计数循环，_ 作丢弃变量
   - while：条件循环
   - break 终止循环，continue 跳过本轮
   - 嵌套循环：九九乘法表示例，内层 range 依赖外层变量
   - time.sleep()：暂停执行，控制输出节奏

5. 综合实例：Craps 赌博游戏（分支和循环实例.py）
   - random.randrange() 生成随机数模拟骰子
   - 嵌套 while 实现游戏主循环 + 下注校验
   - 完整的状态流转：下注 → 摇骰 → 胜负判定 → 资产变化
   - if/elif 实现游戏规则（首轮 7/11 胜，2/3/12 负，其余进入循环）
   - f-string 动态输出当前资产和点数

2026-05-20 学习重点总结
=========================

1. 列表的创建（列表_1.py）
   - 字面量创建：a = [1, 2, 3]，元素可混合类型
   - list() 构造：list(range(1, 6))、list('hello')
   - 列表生成式（列表_2.py）：[i for i in range(10)]、[i**2 for i in range(10)]
   - 嵌套列表生成式：[[random.randint(1,100) for i in range(5)] for j in range(3)]

2. 列表运算（列表_1.py）
   - 加法 +：拼接两个列表
   - 乘法 *：重复列表元素
   - in / not in：成员检测
   - == vs is：== 比较值是否相等，is 比较是否同一对象
   - 大小比较：按元素逐个比较（字典序）

3. 索引和切片（列表_1.py）
   - 正向索引 0 ~ N-1，反向索引 -N ~ -1
   - 切片语法：[start:end:step]，省略 start 从头开始，省略 end 到末尾
   - a[::2] 取偶数位元素，a[::-1] 可反转列表
   - 通过切片可批量修改元素：a[::2] = [0, 0]

4. 列表遍历（列表_1.py）
   - for i in a：直接遍历元素
   - for i in range(len(a))：通过索引遍历

5. 列表的增删操作（列表_2.py）
   - append(x)：在末尾添加元素
   - insert(i, x)：在指定位置插入元素
   - remove(x)：删除第一个匹配的元素（只删一个）
   - pop(i)：删除并返回指定位置元素，默认删除最后一个
   - del list[i]：删除指定位置元素
   - clear()：清空列表

6. 位置查找与统计（列表_2.py）
   - index(x, start)：查找元素首次出现的索引，可指定起始查找位置
   - count(x)：统计元素出现次数

7. 排序与反转（列表_2.py）
   - sort()：默认升序排序（就地排序）
   - reverse()：反转列表顺序

8. 列表嵌套（列表_2.py）
   - 二维列表：[[1,2,3], [4,5,6], [7,8,9]]
   - 通过双重索引访问：list[行][列]
